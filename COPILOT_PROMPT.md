# Copilot Jumpstart 指示書

## 使い方

1. GitHub.com → **New repository**
2. Repository name: `yotei-legends-wiki`
3. **Public** リポジトリ
4. License: なし（後でCC BY-SA 4.0をREADMEとLICENSEで明記）
5. 「**Jumpstart your project with Copilot**」をクリック
6. 下記「プロンプト本文」を貼り付けて生成

```markdown
# 冥人奇譚（Legends）攻略wiki

Ghost of Yōtei の無料DLC「冥人奇譚（Legends）」の日本語攻略wikiです。

!!! info "配信情報"
    - **配信日**: 2026年3月11日（無料DLC）
    - **レイド（大禍）**: 2026年4月配信予定
    - **プラットフォーム**: PS5（PS Plus加入が必要）
    - **アップデートサイズ**: 約26GB

## 冥人奇譚とは

本編のストーリーが「数世紀後に語り継がれた怪談」として再構成されたマルチプレイモード。
本編の羊蹄六人衆が誇張・神話化され、身長4〜5mの鬼ボスとして登場する。（PS公式）

各ボスには固有の配下勢力・サブボスがおり、ボスのテーマ能力に連動。

- 例：狐（Kitsune）配下に Snow Woman（雪女、冷気使いエリート）
- 例：蛇（Snake）配下に召喚師

## クラス一覧

| クラス | フォーカス武器 | 立ち回りイメージ |
|--------|--------------|----------------|
| [侍（Samurai）](classes/samurai.md) | 大太刀（Odachi） | 前線維持・奥義火力 |
| [弓取（Archer）](classes/archer.md) | 槍（Yari） | 遠距離殲滅・エリート処理 |
| [用心棒（Mercenary）](classes/mercenary.md) | 二刀（Dual Katanas） | 投げデバフ・サポート火力 |
| [忍（Shinobi）](classes/shinobi.md) | 鎖鎌（Kusarigama） | 暗殺・ギミック処理 |

- 各クラスは独自のテックツリーを持つ（PS公式）
- フォーカス武器以外も装備可能（PS公式）
- 全クラスで銃（種子島）が装備可能（5ch実測）
- 気力消費で体力回復が可能（本作新要素）（5ch実測）

## 難易度

| 難易度 | 推奨気 | 備考 |
|--------|--------|------|
| 赤銅 | — | 入門。ソロ練習向き |
| 白銀 | 低〜中 | ここから野良マルチが本格化 |
| 黄金 | 65 | 絶品・神品ドロップ開始。推奨気は80以上が実態 |
| 白金 | 80〜 | 高難度。寄生プレイヤーが多く野良は不安定 |
| 百鬼 | 80＋白金クリア | 最高難度。試しの符が強制付与 |

!!! warning "推奨気について"
    公式の推奨気は実態より低め。黄金の推奨65に対し、80以上ないと野良では厳しいとの報告多数。（5ch実測）

## 進行ガイド（ざっくり）

1. **悪鬼成敗（奇譚）** を赤銅で一通りクリア → クラス解放・操作習熟
2. 奇譚3セットクリアで **攻城** が解放 → ボス戦
3. **九死** の赤銅〜白銀で装備・気レベルを上げる
4. 黄金帯で絶品・神品を掘る
5. 4月の **レイド（大禍）** に備える

→ 詳細は [初心者ガイド](guide/beginner.md) へ
```

```markdown
# クラス一覧

冥人奇譚には4つのクラスがある。各クラスはフォーカス武器と専用テックツリーを持つが、
フォーカス武器以外の装備も可能。装備は全クラスで共有できる。（PS公式）

## 比較表

| クラス | フォーカス武器 | 固有能力の方向性 | 九死での役割 |
|--------|--------------|----------------|-------------|
| 侍 | 大太刀 | 前線火力・八幡の怒り（範囲奥義） | 前線維持・ラストウェーブの切り札 |
| 弓取 | 槍 | 遠距離殲滅・内経の眼（ロック射撃） | エリート処理・火力の柱 |
| 用心棒 | 二刀 | 武器投げデバフ・バフ | サポート火力・デバフばらまき |
| 忍 | 鎖鎌 | 闇討ち・分身・霧隠れ | ギミック処理・暗殺 |

## クラス選びの指針

- **初心者向け**: 侍（シンプルに強い、大太刀で剛兵も楽）or 用心棒（投げ物デバフが使いやすい）
- **FPS得意な人**: 弓取（ヘッドショットが全て。装備揃うと最強だが初期は弱い）
- **やり込み向け**: 忍（育つまで弱い。格20の固有能力解放が必須級）

!!! tip "格20解放が重要"
    全クラス共通で、格20で解放される固有能力が必須級。（5ch実測）
    黄金九死以上に行く前に格20まで育てるのが推奨。
```

```markdown
## 特徴

- ヘッドショット精度が全ての職人キャラ
- 神品「水切の長弓」入手で世界が変わる（HSで近くの敵に反射）
- 長弓は引き絞りの有無でダメージが変わらない（パシャ撃ちが有効）（5ch実測）
- 格上がると「HSで矢消費なし」が解放され弾切れ問題が解消

!!! danger "3/14アプデでナーフ"
    True Aim Yumi（必中の長弓）の奥義ゲイン獲得量が大幅に弱体化。
    それでも長弓火力自体は十分高いまま。（5ch実測・パッチノート）

## 弾薬管理の重要性

- **飛び道具補給は全員共有リソース**（5ch実測。前作は個別だった）
- 長弓は10本しかない。序盤は近接で節約し、終盤のエリート大群に火力を集中
- 弓取がいる場合、他クラスは弾薬を譲るのが基本マナー（ただし強制ではない）

## 九死での注意

- 高台から撃つだけで陣地に降りない弓取は地雷扱いされる
- 陣地防衛時間が極端に短い＝チームの足を引っ張っている
- 蘇生も重要な仕事。目潰し玉で安全に蘇生できる

## 装備方針

<!-- TODO: 具体的なビルド例を追記 -->

- 奥義回復を全装備に盛る → 内経の眼の回転率が段違いに
- 半弓（サブ）はオートエイム付きの名品が便利
- 撒菱で弾薬ドロップを確保する構成もあり
```

```markdown
# 九死（サバイバル）

## 基本ルール

- **4人協力**のウェーブ防衛モード（PS公式）
- マップ上の拠点を守りながら、次々と押し寄せる敵を捌く
- 4マップ（蜘蛛・鬼・狐・蛇に対応）（PS公式JP）

## 拠点システム

### 拠点を維持するメリット

- 拠点内にいると防衛ゲージが維持される
- 「祝福」効果が発動（マップごとに異なる）

### 拠点を落とされたペナルティ

- プレイヤー全員の**最大HPが減少**（5ch実測）
- **影の敵**が召喚される（5ch実測）
- 鉄砲隊が陣地にロックして攻撃してくる場合あり
- 落ちた陣地に残って戦い続けるのは無意味。即座に別の陣地へ移動すべき（5ch実測）

## 飛び道具補給は全員共有

!!! danger "最重要：前作との違い"
    **飛び道具の補給オブジェクトは全員共有リソース**。（5ch実測）
    前作（対馬）は個別だったが、今作は1つの補給を全員で分け合う。
    4段階×4回分。1本だけの補充でも1回分を消費する。

- 弓取がいる場合は優先的に譲るのが基本マナー
- 序盤は近接で節約し、終盤のエリート大群に備える

## 大岩加護（拠点購入）

各拠点には加護の購入ポイントがある。（5ch実測）

### 全体効果（使用時に全員に反映）

- 体力＆気力回復
- 飛道具補充＆威力2倍

### 個人効果（購入者のみ）

- マップ固有の能力（例：狐マップのワープ能力）
- 個人用は大岩加護の裏側にある（5ch実測）

!!! tip "狐マップのワープ"
    狐の九死ではワープ能力の購入がほぼ必須。
    購入場所は大岩加護の裏の個人用ショップ。（5ch実測）

## 貢ぎ物（勾玉）

- 勾玉の収集は**個人別**（共有ではない）（5ch実測）
- 敵を倒すと勾玉がドロップ
- 集めた勾玉で加護を購入

## 試しの符

難易度を上げる代わりに報酬が増える縛りカード。

### 比較的安全な符

- ☆2の「敵の数増加」「敵攻撃速度UP」系は、立ち回りでカバーしやすい

### 危険な符（野良では避けるべき）

| 符 | 効果 | 危険度 |
|----|------|--------|
| 死体爆発 | 敵が死ぬと爆発。近接職が事故死しやすい | :material-alert: 高 |
| 飛び道具でのみキル可能 | 近接でトドメが刺せない | :material-alert: 高 |
| 味方とダメージ共有 | 一人が被弾すると全員にダメージ | :material-alert: 高 |
| 味方から離れるとダメージ | 陣地分散が困難に | :material-alert: 高 |
| 蘇生時間延長 | 蘇生がほぼ不可能に | :material-alert: 高 |
| 体力徐々に減少 | 常時HP減少。体力回復購入必須 | :material-alert: 中 |
| 閃光弾 | 視界を奪われる | :material-alert: 中 |
| 剛兵追加 | 火力的にはきついが対処は可能 | 中 |

!!! warning "試しの符のマナー"
    推奨気を大幅に下回るプレイヤーが高難度の符を選ぶのは地雷行為。
    九死では完走できないリスクがクリア報酬を上回る場合が多い。（5ch実測）

## マップ別ギミック

### 蛇マップ（3色ギミック）

- 敵が3色の精霊を纏って出現
- 対応する色の加護を灯篭で切り替えて攻撃する必要がある
- 色を合わせないとダメージが通らない
- 理解していないプレイヤーが多いが、合わせれば最も低難度のマップとの報告（5ch実測）

### 狐マップ（ワープ＋雪女）

- ワープ能力の購入が必須
- 雪女はHP減少＋ワンパン即死の危険性
- 雪女が消えたら3体の氷像の中に隠れている→全部破壊で出現（5ch実測）
- 火属性が有効

### 鬼マップ（風神雷神ギミック）

- 風神と雷神の加護を2人で別々に使う協力ギミック
- 風神3人は欲しい。安定してダメージパートに移行するため（5ch実測）
- ソロだときつい

### 蜘蛛マップ

- 時限弾の回避が難所
- 超範囲攻撃の避け方が分かりにくいとの報告（5ch実測）

## 野良マナー

- 推奨気以上で参加する（推奨気は目安より+15〜20が実態）
- 拠点の基本配置は2:2。押されたら2:1:1で1側は生存重視
- 弓取に飛び道具補給を優先的に譲る
- 落ちた拠点に居座らない→即座に別拠点へ移動
- 試しの符は無理に高難度を選ばない
```

```markdown
# 装備システム

## 気（Ki）レベル

- 装備全スロットの平均値が「気レベル」になる（前作同様）
- 気レベルが高いほど、ドロップする装備のレベルも高くなる
- **現状の上限は110**（5ch実測。アプデで上限解放の可能性あり）

## 装備ランク

| ランク | 気上限 | 入手先 |
|--------|--------|--------|
| 良品 | 低 | 赤銅〜白銀 |
| 名品 | 中 | 白銀〜 |
| 絶品 | 80程度 | 黄金〜 |
| 神品（レジェンダリー） | 上限なし | 黄金〜（低確率） |

## 気の強化

- 素材を使って装備のレベルを上げられる
- **コストが安いのでケチらずやるべき**（5ch実測）
- 強化してレベルを上げる→ドロップのレベルも上がる→効率的に成長
- 良品→名品のような「位を上げる」強化は大量の素材を消費するので注意

!!! tip "公式チュートリアルの助言は正しい"
    「最上級を手に入れるより武具強化した方がいい」というチュートリアルは正しい。
    序盤からコツコツ強化して平均気を上げていくのが最効率。（5ch実測）

## 特殊効果判定（重要）

!!! danger "厳選の核心"
    装備には **特殊効果判定** のランクがある（D〜S）。
    **ドロップ時に固定** され、後から変更するには大量の素材が必要。

- 2つの判定があり、例えば「A-」と「B+」のように表示される
- 気レベルが低い装備でもA判定が出ることがある
- **分解する前に判定を確認すべき**（5ch実測で多くの人がA判定装備を捨てて後悔）

### 判定の上げ方

- 強化で判定を上げることは可能（C→C+のように段階的）
- ただし絶品の判定強化は**結晶200個**と高コスト（5ch実測）
- 結晶200個 = 神品強化10回分 = 絶品強化100回分
- ADのような偏った装備より、BBのように平均の高い装備を強化する方が効率的

### 判定の上限

- Sで上限。片方がSになるとそれ以降はもう片方のランクしか上がらない
- 一部の判定（奥義回復など）はA手前とSで数値が変わらない場合あり（5ch実測）

## 装備ドロップの仕様

- 装備は**クラス依存のドロップテーブル**がある
- 例：忍では鎖鎌が落ちやすいが、侍では落ちない
- 他クラスの装備も低確率で出現する場合あり（5ch実測）
- 装備は全クラスで共有可能
```

```markdown
# 気レベル上げ・周回

## 基本方針

1. **気の強化をケチらない**（素材は安い。序盤から積極的に強化）
2. 低い装備スロットを重点的に上げる（平均値なので偏りを解消）
3. 強化→ドロップレベル上昇→さらに強い装備入手のサイクルを回す

## 効率的な周回先

| 気レベル帯 | おすすめ | 理由 |
|-----------|---------|------|
| 〜50 | 赤銅〜白銀 奇譚 | ソロで高速周回可能 |
| 50〜80 | 黄金 奇譚（蜘蛛壱が最短） | マッチング速い、短時間 |
| 80〜100 | 黄金 九死 or 攻城 | 神品ドロップあり |
| 100〜110 | 百鬼 | 白金より安定する場合も |

!!! tip "奇譚ソロ周回"
    マッチング待ち時間がないため、ソロ奇譚周回が気上げには効率的。
    弓取で白金奇譚を回すと高速。（5ch実測）

## 素材について

- 装備のレアリティで入手素材が決まる
- 絶品入手で全素材が手に入る
- 分解でも素材は得られる（アプデで報酬増加済み）
- 技解放用の素材（クリスタル・石）は黄金以上の報酬で入手（5ch実測）
```
