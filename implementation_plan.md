# 実装プラン：特殊効果判定（品格）の名称変更・統一

## 目的
ゲーム内の正式名称である「品格」を取り入れつつ、ユーザーにとって直感的に分かりにくいという課題を解決するため、意味が伝わりやすい表現と併記する形でWiki内の記述を修正する。

## 修正対象ファイル
1. `docs/gear/index.md`
2. `docs/guide/beginner.md`

## 提案する選択肢（ユーザーへの承認待ち）

Wikiとしての分かりやすさと、実際のゲーム画面との整合性をどう取るかについて、以下の2案を提案します。

**案A：分かりやすさ優先（当エージェント推奨）**
初心者への伝わりやすさを重視し、直感的な「特殊効果判定」をメインに据え、カッコ書きで公式名称を入れる。
* 修正例:
  - 見出し: `## 特殊効果判定（ゲーム内名称：品格）`
  - 本文: `装備には **特殊効果判定（品格）** のランクがある（D〜S）。`

**案B：ゲーム内名称優先**
実機表記に合わせることを重視し、「品格」をメインに据え、カッコ書きで意味を補足する。
* 修正例:
  - 見出し: `## 品格（特殊効果判定ランク）`
  - 本文: `装備には **品格（特殊効果の強さを表すランク）** がある（D〜S）。`

## 実装ステップ
1. ユーザーから案A・案B（あるいは別の表現）のどちらにするか回答を得る。
2. 回答に基づき、`docs/gear/index.md` および `docs/guide/beginner.md` の該当箇所を一括置換する。
3. ローカルのMkDocsサーバーで表示崩れがないか確認する。
