    ステージギミック解説¶

    冥人奇譚の各ステージには独自のギミックが存在する。 ギミックを理解することがクリアへの近道。野良マルチでもギミックを把握していると大きなアドバンテージになる。

    🔥 鬼火（Oni-bi）とは¶

    鬼火（おにび）は、冥人奇譚の各ステージに現れる怨霊・霊力の炎のこと。

    ゲーム内での出現パターン¶

    形態 説明
    環境ハザード ステージ上に浮かぶ青白い炎のオブジェクト。触れると持続ダメージ・呪い状態
    敵に纏う炎 一部の霊的エリート敵が鬼火を纏っており、近づくと燃焼ダメージ
    ゴースト武器 「鬼火の放心玉（Vengeful Onibi Bomb）」として武器化。爆発スタン

    鬼火の特性¶

    • 青白く発光しており、暗いステージでは視覚的な目印になる
    • 霊属性の攻撃・能力と組み合わさると効果が増幅する場合あり（要検証）
    • 鬼火を纏った敵は物理ダメージが軽減されることがある → まず鬼火を剥がす
    • 忍の「霧隠れ」中は鬼火のハザードを無効化できるとの報告あり（5ch実測・要検証）

    鬼火の放心玉（Vengeful Onibi Bomb）¶

    神品ゴースト武器

    • 効果: 投擲時に鬼火の爆発が発生し、範囲内の敵をスタンさせる
    • 入手: 神品ドロップ（九死・攻城の黄金以上推奨）
    • 放心玉カテゴリの神品。通常の放心玉より範囲・スタン時間が長い
    • エリート敵・ボスの怯み役として有用

    🏯 陣（拠点）を占領されると…¶

    九死モードでは、マップ上に複数の陣（拠点）が存在し、これを守ることが勝利の鍵。 一つでも占領されるとチーム全体に深刻なペナルティが発生する。

    陣を落とされた際のペナルティ一覧¶

    ペナルティ 詳細
    最大HP減少 プレイヤー全員の最大HPが一定量減少する（5ch実測）
    影の敵召喚 特殊な「影の敵」が呼び出される。通常の敵より強力（5ch実測）
    鉄砲隊によるロック攻撃 占領された陣地に鉄砲隊が展開し、陣内にいるプレイヤーを集中砲撃（5ch実測）
    防衛ゲージへの影響 守れる陣の数が減り、防衛ゲージの維持が困難になる
    祝福効果の喪失 陣に付随する「祝福」バフが消える（マップごとに異なる）

    落ちた陣に居座ってはいけない

    占領された陣地に残り続けるのは最悪の選択。鉄砲隊の集中砲火を受けるうえ、 他の陣地の防衛人数が減るためチーム全体の崩壊を招く。 占領されたら即座に隣の陣地へ移動するのが鉄則。（5ch実測）

    陣を守るコツ¶

    • 基本配置は 2:2（4人で2陣地を均等に守る）
    • 押されてきたら 2:1:1 に移行し、1人は生存優先で動く
    • 陣の「大岩加護」で全体回復・弾薬補充を購入してゲームを有利に進める
    • 陣の祝福効果をフル活用するために常に陣内を意識する

    🕷️ 蜘蛛ステージのギミック¶

    時限弾（タイムボム）¶

    • ステージ中に時限弾が設置される場面がある
    • 時限弾は爆発前にオレンジ色の光を放つ → 範囲から離れる or 忍が処理
    • 超広範囲の爆発が特徴。慌てて逃げると壁に詰まって被弾しやすい

    超範囲爆発の回避

    爆発範囲は見た目より広い。光ったら早めに距離を取ること。 蜘蛛の攻城ボスも同様の攻撃をしてくる。（5ch実測）

    👹 鬼ステージのギミック¶

    風神・雷神の加護¶

    • ギミック概要: ステージ内に「風神の加護」「雷神の加護」の2種類が存在
    • 2人で別々の加護を使って協力することで特殊フェーズに移行できる設計
    • 一人が両方取ると全く噛み合わない場合あり → 役割分担を意識する

    推奨構成

    風神3人分 を取ってから攻撃フェーズに移行すると安定するとの報告。 ソロだと非常に困難なギミック。（5ch実測）

    • 風神の加護：移動速度UP・一部ボス攻撃の回避が容易に
    • 雷神の加護：攻撃力UP・特定エリアへのダメージ増加（要検証）

    🦊 狐ステージのギミック¶

    雪女（ゆきおんな）¶

    • 狐ステージのエリートエネミー。冷気属性の強敵
    • 接触・攻撃で最大HPが継続的に減少し、最悪ワンパン即死ライン到達
    • 倒し方: 雪女が消えたら、3体の氷像の中に隠れている → 全部破壊すると出現（5ch実測）
    • 有効属性: 火属性。火のない状態では壊滅的に不利

    火属性なしは危険

    狐の攻城・九死において、パーティに火属性がない場合は雪女処理が非常に困難。 焔の剣（炎エンチャント）や炎系パークを確保しておくと安心。（5ch実測）

    ワープ能力¶

    • 狐の九死マップではプレイヤーがワープ能力を購入できる
    • 購入場所は大岩加護の裏側にある「個人用ショップ」（5ch実測）
    • 広いマップを移動するのにほぼ必須。購入を忘れると陣地移動に大幅な時間を取られる

    🐍 蛇ステージのギミック¶

    3色精霊システム（最重要ギミック）¶

    このギミックを理解するだけでクリア率が大幅UP

    蛇ステージ最大のギミック。理解していないプレイヤーが多いが、 把握すれば全ステージ中最も難易度が低いとの報告もある。（5ch実測）

    仕組み：

    1. 敵が 赤・青・黄（緑）の精霊を纏って出現
    2. マップ上の灯篭でプレイヤーの加護の色を切り替える
    3. 敵の色と加護の色を合わせることでダメージが通る
    4. 色が合っていないとほぼダメージが入らない

    立ち回り方：

    • 戦闘前に灯篭の位置を確認する
    • 敵の色を見てから、全員で同じ灯篭に切り替えるか、色ごとに担当を分ける
    • 2:2（2色を2人ずつ担当）の分担が安定しやすい（5ch実測）

    ⚔️ 共通ギミック¶

    精霊付きの敵¶

    • 特定の敵が霊的なオーラ（精霊）を纏った状態で出現
    • この状態では物理ダメージが大幅に軽減される
    • 対処法: まず物理攻撃（特に鎖鎌が便利）で精霊を剥がす → 本体を攻撃（5ch実測）

    穢れ（Corruption）¶

    • 紫色の範囲エフェクトが出現する場所。一定時間いると大ダメージ〜即死
    • ステージ進行中に急に広がることがある → 常にマップを確認
    • 穢れゾーンに引き込もうとする敵の行動パターンあり（要検証）

    樹木子（じゅもくこ）¶

    • 赤い玉（腫瘍）を体に持った植物系エネミー
    • 対処法: 腫瘍（赤い玉）を遠距離攻撃で破壊 → 無力化 → 倒す（5ch実測）
    • 破壊せずに近接だけで戦おうとすると大ダメージを受ける場合あり

    ボス共通：黄色の溜め攻撃¶

    • ボスが黄色いオーラを纏って溜め攻撃を始めた場合、体幹ゲージを削り切るとキャンセル可能（5ch実測）
    • タイミングよくカット/割り込みを入れられるかどうかでパーティの生存率が大きく変わる
    • フェーズ移行はカメラワーク・環境色の変化が目印
