# 更新履歴

## ゲームアップデート

### v1.512（2026/3/20）

**一般**

- NG+（シングルプレイ）セーブデータの一部コスメがLegendsでアンロック
- パーティメンバー切断時、フレンド招待やマッチメイキングで空き枠を埋めるよう促すプロンプトの追加
- Legendsに入った後、一部のプレイヤーで旅人の地図（Traveler's Maps）が消えるバグを修正
- 百鬼（Nightmare）ミッションを最後までクリアせずに報酬を得られていたバグを修正
- Legendsプレイ時にロードアウトが消去されるバグを修正
- Shadow Strike Featのアンロック条件が明确になるよう説明文を修正

**ゲームプレイ**

- Legendsモードの安定性向上
- マッチメイキングに必要最低気レベル（Ki levels）を追加
- コンテンツ難易度に合わせて推奨気レベルを調整
- 全クラスのVitality IおよびVitality IIスキルを調整
- 進行のより早い段階で体力を増やせるよう、アンロックランクを調整
- 侍（Samurai）のVitality Iの効果を増加

### v1.511（2026/3/14）

**一般**

- リーダーボード画面に追加情報を導入
- リーダーボード用のアクティブなチャレンジカードを追加
- スコア入力までの残り時間タイマーを追加
- 忍のマスクが他のコスメと重なる問題を修正（デフォルトがスキンヘッドに、頭巾は装具扱い）
- 外したデフォルトのコスメやエモートを再装備可能に変更
- サバイバルミッションのクイックプレイがデフォルト解放
- プレイヤーと敵の挙動に関するオーディオおよびハプティクスの調整
- その他のバグ修正と安定性向上
- BGM途切れバグ修正
- 鬼尾撃破時のアニメーションバグ修正
- Frozen Valley生存ミッションの武器庫防衛位置の可視性向上

**ゲームプレイ**

- Spirit Kunaiが他のKunai武器パークと重複するよう修正
- Caltrops・Spirit・奥義チャージ・弾薬のドロップ率調整
- 装備分解時の通貨報酬増加
- 風神の風の応答性向上
- **True Aim Yumiの奥義ゲイン大幅弱体化**
- Jufuの近接ターゲットを調整
- Shadow Flame Arrowの炎上バグ修正
- 蘇生ターゲットの改善
- てるてる坊主操作後のプレイヤー操作不能バグ修正

### v1.500（2026/3/11）

- 冥人奇譚（Legends）配信開始
