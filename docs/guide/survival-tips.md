# 九死の立ち回りガイド

**九死（サバイバル）** モードで安定してクリアするための立ち回りを解説します。

## 九死の基本

九死は複数のウェーブを生き延びるモードです。全ウェーブを生き残ることでクリアとなります。

!!! warning "要検証"
    ウェーブ数・構成の詳細は調査中です。

## 成功のカギ

### 1. ポジショニング

!!! tip "陣形を維持する"
    - **後ろに下がりすぎない** — 蘇生できないエリアに逃げ込まないように注意
    - **前に出すぎない** — 囲まれると脱出困難になる
    - **チョークポイントを活用** — 狭い通路を使って敵の横広がりを防ぐ

### 2. 優先ターゲット

敵には優先度があります。以下の順番で倒すのが基本です：

1. **回復役・支援役の敵** — 放置すると敵がどんどん回復する
2. **遠距離攻撃の敵** — 安全な場所から継続ダメージを与えてくる
3. **エリート敵** — 攻撃力が高く早めに処理したい
4. **雑魚敵** — 最終的には全員倒す

### 3. スキル管理

!!! tip "スキルは惜しみなく使う"
    スキルはクールダウン制のため、ウェーブ間で回復します。温存しすぎず積極的に使いましょう。

### 4. 蘇生

!!! warning "蘇生は最優先！"
    倒れた味方は迅速に蘇生しましょう。人数が減ると一気に形勢不利になります。蘇生中は隙があるので、他のメンバーが敵を引き付けましょう。

## クラスごとの役割

=== "侍"

    - 前衛として敵の攻撃を引き付ける
    - 「不動の構え」を大技に合わせて使う
    - チョークポイントで敵の進路を塞ぐ

=== "弓取"

    - 後方から範囲攻撃で雑魚を掃討
    - 「爆裂矢」でエリート敵を優先処理
    - 煙幕で近づいてきた敵をやり過ごす

=== "用心棒"

    - 高い攻撃力で素早く敵を排除
    - エリート敵・ボスに「必殺の一撃」を使う
    - 倒れた仲間の近くで敵を引き付ける

=== "忍"

    - 敵の注意を分散させる「分身の術」を活用
    - 孤立した敵を「影走り」で奇襲する
    - 蘇生の際の護衛役として機能する

## おすすめ構成別戦術

### 防衛型構成（侍2 + 弓取1 + 用心棒1）

長時間の戦闘に耐えられる安定構成。

- 侍2人が前線を支えながら弓取・用心棒が後方支援
- 侍は互いにバックアップしてダウン時間を最短化

### 高速クリア型（用心棒2 + 弓取2）

高い火力で素早くウェーブをクリアする攻撃的構成。

- 耐久力は低いので被弾しないことが重要
- エリート敵は全員で集中攻撃して早期排除

## よくある失敗例

!!! warning "やりがちなミス"
    - **蘇生に行けずジリ貧になる** — 早めに声をかけて蘇生役を決める
    - **ボスに集中しすぎて雑魚が増える** — 雑魚処理も並行して行う
    - **スキルを温存しすぎて全滅** — スキルはその場その場で最大限活用する
