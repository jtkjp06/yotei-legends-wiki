# 九死の立ち回り

→ 基本ルールは [九死](../modes/survival.md) を参照

## 配置の基本

- **基本は2:2**で拠点を分担
- 押されたら**2:1:1**。1人側は生存重視で味方の合流を待つ
- 4人全員が1拠点に集まるのはNG（他の拠点が無防備に）

## クラス別の動き方

### 侍
- 前線で穴埋め。死なないこと＞キル数
- 八幡の怒りはラストウェーブの切り札

### 弓取
- エリート・ボス・大筒持ちを優先的に処理
- **陣地内にも降りる**。高台に籠りっぱなしはNG
- 飛び道具は計画的に使う

### 用心棒
- 投げデバフで味方を支援
- 単独での陣地防衛は火力不足になりがち

### 忍
- ギミック処理（暗殺必須の敵、特定処理）を担当
- 目潰し→闇討ちコンボで効率的に処理
- 分身でヘイト分散

## よくある失敗と対策

| 状況 | 対策 |
|------|------|
| 落ちた陣地で戦い続けて死亡 | 即座に別拠点へ移動 |
| 弾薬枯渇 | 序盤は近接で節約。終盤に備える |
| 低レベルプレイヤーとマッチ | 開始前にメンバーの気を確認。厳しければ抜ける |
| ギミックが分からない | 奇譚を先にクリアして予習 |
| 味方の蘇生が間に合わない | 目潰し玉で安全確保してから蘇生 |
