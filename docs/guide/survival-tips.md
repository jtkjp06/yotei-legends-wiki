# 九死の立ち回り

→ 基本ルールは [九死](../modes/survival.md) を参照

## 配置の基本

- **基本は2:2**で拠点を分担
- 押されたら**2:1:1**。1人側は生存重視で味方の合流を待つ
- 4人全員が1拠点に集まるのはNG（他の拠点が無防備に）
- 推奨気は目安より+15〜20が実態。推奨未満だと火力も耐久も足りない[（5ch報告）](../sources/5ch-threads.md)

## クラス別の動き方

### 侍
- 前線で穴埋め。死なないこと＞キル数
- 八幡の怒りはラストウェーブの切り札

### 弓取
- エリート・ボス・大筒持ちを優先的に処理
- **陣地内にも降りる**。高台に籠りっぱなしはNG
- 飛び道具は計画的に使う

### 用心棒
- 投げデバフで味方を支援
- 単独での陣地防衛は火力不足になりがち

### 忍
- ギミック処理（暗殺必須の敵、特定処理）を担当
- 目潰し→闇討ちコンボで効率的に処理
- 分身でヘイト分散

## よくある失敗と対策

| 状況 | 対策 |
|------|------|
| 落ちた陣地で戦い続けて死亡 | 即座に別拠点へ移動 |
| 弾薬ドロップの奈れ合い | 序盤は近接で節約。終盤に備える |
| 低レベルプレイヤーとマッチ | 開始前にメンバーの気を確認。厳しければ抜ける |
| ギミックが分からない | 奇譚を先にクリアして予習 |
| 味方の蘇生が間に合わない | 目潰し玉で安全確保してから蘇生 |
| 大筒鬼（青い鬼）に爆殺される | 最優先で処理。精度が高く即死級の火力[（5ch報告）](../sources/5ch-threads.md) |

## 前作との主な違い

- **受け流しキャンセル**（受け流しから即蘇生等への繋ぎ）が今作ではできない模様[（5ch報告）](../sources/5ch-threads.md)
- **気力を使って体力回復**がマルチでも使用可能に。回復の香の価値は前作より下がっている[（5ch報告）](../sources/5ch-threads.md)
- **放心玉の弱体化**: よろめき時間が前作より短く、使い勝手が悪い[（5ch報告）](../sources/5ch-threads.md)
- **敵の攻撃範囲が広い**: カメラ外から攻撃してくる仕様。前作の「視界に入っていない敵は攻撃してこない」が通用しない[（5ch報告）](../sources/5ch-threads.md)
