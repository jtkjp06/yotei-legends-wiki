# 気レベル上げ・周回ガイド

**気（き）レベル**はキャラクターの装備強度を示す総合指標です。このガイドでは気レベルを効率的に上げる方法を解説します。

## 気レベルとは

気レベルは装備しているすべての装備品のレベルの平均値（または合計）で決まります。

!!! warning "要検証"
    気レベルの計算式は現在調査中です。

## 気レベルの上げ方

### 基本的な方法

1. **ミッションをクリアして装備を集める**
2. **入手した装備に付け替えてレベルを更新する**
3. **繰り返してレベルを上げていく**

    - **高難度ほど良い装備が出やすい** — 自分の気レベルで攻略可能な最高難度に挑戦しよう

    - **神品狙いは大禍が最効率** — ある程度気レベルが上がったら大禍への移行を推奨

## クラスごとの周回適性

| クラス | 周回適性 | コメント |
|--------|---------|--------|
| 侍 | ★★★★☆ | 安定した立ち回りで被弾が少ない |
| 弓取 | ★★★★★ | 範囲攻撃で敵を素早く処理できる |
| 用心棒 | ★★★★★ | 高い攻撃力でスムーズに進められる |
| 忍 | ★★★☆☆ | 低耐久が若干のリスクになる |

## まとめ

1. 自分の気レベルに合った最高難度のコンテンツを選ぶ
2. クリアして装備を収集し、装備を更新する
3. 上位コンテンツへ移行する
4. 繰り返す

!!! note "焦らずに"
    気レベル上げは時間がかかります。焦らず着実に上げていきましょう。
