# 九死（サバイバル）

**九死（くし）** は波状の敵攻撃を生き延びる**サバイバルモード**です。複数のウェーブを乗り越えることで報酬を獲得できます。

## モード概要

- **人数**: 1〜4人
- **目的**: 指定ウェーブをすべて生き延びる
- **特徴**: ウェーブごとに強化オプションを選択できる

## 立ち回りガイド

### 序盤（ウェーブ 1〜3）

!!! tip "序盤の心がけ"
    - 体力・スタミナのリソースを温存する
    - 敵の出現位置を把握して適切なポジショニングをとる
    - 強化オプションは長期戦を見据えて選ぶ

### 中盤（ウェーブ 4〜6）

- エリート敵が出現しはじめる
- 集中して優先順位の高い敵（遠距離攻撃や回復役）を先に倒す

### 終盤（ウェーブ 7〜）

- ボスクラスの敵が出現する
- 全員が連携して集中攻撃をしかける

詳細は [九死の立ち回り](../guide/survival-tips.md) ガイドも参照してください。

| 構成 | 特徴 |
|------|------|
| 侍2 + 弓取1 + 忍1 | 安定した防衛ライン |
| 用心棒2 + 弓取2 | 高火力で素早く敵を排除 |
