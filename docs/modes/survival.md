# 九死（サバイバル）

## 基本ルール

- **4人協力**のウェーブ防衛モード（PS公式）
- マップ上の拠点を守りながら、次々と押し寄せる敵を捌く
- 4マップ（蜘蛛・鬼・狐・蛇に対応）（PS公式JP）

## 拠点システム

### 拠点を維持するメリット

- 拠点内にいると防衛ゲージが維持される
- 「祝福」効果が発動（マップごとに異なる）

### 拠点を落とされたペナルティ

- プレイヤー全員の**最大HPが減少**（5ch実測）
- **影の敵**が召喚される（5ch実測）
- 鉄砲隊が陣地にロックして攻撃してくる場合あり
- 落ちた陣地に残って戦い続けるのは無意味。即座に別の陣地へ移動すべき（5ch実測）

## 飛び道具補給は全員共有

!!! danger "最重要：前作との違い"
    **飛び道具の補給オブジェクトは全員共有リソース**。（5ch実測）
    前作（対馬）は個別だったが、今作は1つの補給を全員で分け合う。
    4段階×4回分。1本だけの補充でも1回分を消費する。

- 弓取がいる場合は優先的に譲るのが基本マナー
- 序盤は近接で節約し、終盤のエリート大群に備える

## 大岩加護（拠点購入）

各拠点には加護の購入ポイントがある。（5ch実測）

### 全体効果（使用時に全員に反映）

- 体力＆気力回復
- 飛道具補充＆威力2倍

### 個人効果（購入者のみ）

- マップ固有の能力（例：狐マップのワープ能力）
- 個人用は大岩加護の裏側にある（5ch実測）

!!! tip "狐マップのワープ"
    狐の九死ではワープ能力の購入がほぼ必須。
    購入場所は大岩加護の裏の個人用ショップ。（5ch実測）

## 貢ぎ物（勾玉）

- 勾玉の収集は**個人別**（共有ではない）（5ch実測）
- 敵を倒すと勾玉がドロップ
- 集めた勾玉で加護を購入

## 試しの符

難易度を上げる代わりに報酬が増える縛りカード。

### 比較的安全な符

- ☆2の「敵の数増加」「敵攻撃速度UP」系は、立ち回りでカバーしやすい

### 危険な符（野良では避けるべき）

| 符 | 効果 | 危険度 |
|----|------|--------|
| 死体爆発 | 敵が死ぬと爆発。近接職が事故死しやすい | :material-alert: 高 |
| 飛び道具でのみキル可能 | 近接でトドメが刺せない | :material-alert: 高 |
| 味方とダメージ共有 | 一人が被弾すると全員にダメージ | :material-alert: 高 |
| 味方から離れるとダメージ | 陣地分散が困難に | :material-alert: 高 |
| 蘇生時間延長 | 蘇生がほぼ不可能に | :material-alert: 高 |
| 体力徐々に減少 | 常時HP減少。体力回復購入必須 | :material-alert: 中 |
| 閃光弾 | 視界を奪われる | :material-alert: 中 |
| 剛兵追加 | 火力的にはきついが対処は可能 | 中 |

!!! warning "試しの符のマナー"
    推奨気を大幅に下回るプレイヤーが高難度の符を選ぶのは地雷行為。
    九死では完走できないリスクがクリア報酬を上回る場合が多い。（5ch実測）

## マップ別ギミック

### 蛇マップ（3色ギミック）

- 敵が3色の精霊を纏って出現
- 対応する色の加護を灯篭で切り替えて攻撃する必要がある
- 色を合わせないとダメージが通らない
- 理解していないプレイヤーが多いが、合わせれば最も低難度のマップとの報告（5ch実測）

### 狐マップ（ワープ＋雪女）

- ワープ能力の購入が必須
- 雪女はHP減少＋ワンパン即死の危険性
- 雪女が消えたら3体の氷像の中に隠れている→全部破壊で出現（5ch実測）
- 火属性が有効

### 鬼マップ（風神雷神ギミック）

- 風神と雷神の加護を2人で別々に使う協力ギミック
- 風神3人は欲しい。安定してダメージパートに移行するため（5ch実測）
- ソロだときつい

### 蜘蛛マップ

- 時限弾の回避が難所
- 超範囲攻撃の避け方が分かりにくいとの報告（5ch実測）

## 野良マナー

- 推奨気以上で参加する（推奨気は目安より+15〜20が実態）
- 拠点の基本配置は2:2。押されたら2:1:1で1側は生存重視
- 弓取に飛び道具補給を優先的に譲る
- 落ちた拠点に居座らない→即座に別拠点へ移動
- 試しの符は無理に高難度を選ばない
