# 悪鬼成敗（奇譚）

**悪鬼成敗（奇譚）** はメインストーリーを追うゲームモードです。様々なミッションをこなしながらストーリーを進め、新しい装備や能力を解放していきます。

## モード概要

- **人数**: 1〜4人
- **目的**: 各ミッションの目標を達成する
- **報酬**: 装備品、気ポイント
