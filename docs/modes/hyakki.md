# 百鬼（ひゃっき）

**百鬼**は100体の敵を次々と撃破するチャレンジモードです。制限時間内に倒した数が記録されます。

## モード概要

- **人数**: 1〜4人
- **目的**: できるだけ多くの敵（最大100体）を倒す
- **特徴**: スコアアタック要素あり

!!! warning "要検証"
    百鬼の詳細なルールは現在調査中です。

## 攻略のポイント

!!! tip "効率的な敵処理"
    - 範囲攻撃を活用して複数の敵をまとめて倒す
    - 「弓取」の爆発物や「用心棒」の連撃が有効
    - 移動効率を上げるクラス・装備選びが重要

## おすすめクラス構成

| 構成 | 特徴 |
|------|------|
| 弓取2 + 用心棒2 | 範囲攻撃で効率よく処理 |
| 忍2 + 弓取1 + 用心棒1 | スピードと範囲のバランス型 |
