# 百鬼

## 基本情報

- 最高難度。試しの符が**強制付与**される[（5ch報告）](../sources/5ch-threads.md)
- 奇譚・攻城・九死すべてに百鬼版がある

## 解放条件

- **白金クリア**が必要[（5ch報告）](../sources/5ch-threads.md)
- 気レベル80以上が目安[（5ch報告）](../sources/5ch-threads.md)

!!! info "百鬼 vs 白金"
    試しの符が固定されている分、白金で他プレイヤーが地雷な符を選ぶよりも
    百鬼の方が安定するとの報告もある。[（5ch報告）](../sources/5ch-threads.md)

## 難易度の実態

- 慣れたプレイヤーが多く、8割クリアできるとの報告（配信5日目時点）[（5ch報告）](../sources/5ch-threads.md)
- 九死の百鬼（雪女あり）が最も難しい
- 寄生プレイヤーが1人いるだけでクリア不可能
