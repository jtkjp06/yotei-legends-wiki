# 攻城（こうじょう）

**攻城**は敵が支配する拠点を攻略するゲームモードです。侵入、制圧、脱出の3フェーズで構成されています。

## モード概要

- **人数**: 1〜4人
- **目的**: 拠点を攻略し、目標を達成して脱出する
- **特徴**: 3フェーズ構成（侵入・制圧・脱出）

## フェーズ解説

### フェーズ1：侵入

拠点への侵入ルートを選択します。

!!! tip "侵入ルートの選び方"
    - 「忍」クラスがいる場合は隠密ルートが有効
    - 強攻は敵の数が多く消耗しやすい
