# 攻城

## 基本ルール

- **最大4人**で挑む大規模ミッション（PS公式JP）
- 奇譚の各セット（3話）をクリアすると対応する攻城が解放
- ラストに羊蹄六人衆のボス戦

## ボス一覧

| ボス | テーマ | 備考 |
|------|--------|------|
| 蜘蛛 | 時限弾・範囲攻撃 | 超範囲時限弾の回避が難所。「蜘蛛の壱・弐」が稼ぎに向くとの報告も[（5ch報告）](../sources/5ch-threads.md) |
| 鬼 | 風神雷神ギミック | 2人で風神・雷神を分担して祓う等の協力必須？敵に頭がなくヘッドショット判定なし[（5ch報告）](../sources/5ch-threads.md) |
| 狐 | 雪女＋冷気 | 松明を投げて燃やした地面の上で戦うのが正攻法。左入カで狐パワーを使ってワープするギミックあり[（5ch報告）](../sources/5ch-threads.md) |
| 蛇 | 3色ギミック | 同じ属性（色）で殴らないとダメージが通らないため持込属性に注意。弓の出番が減りがち[（5ch報告）](../sources/5ch-threads.md) |

!!! info "その他の強敵・ギミック"
    - **樹木子**: 腫瘍が無い時は被ダメージが通らない。蕾を投げて当てると腫瘍等が吹き飛ぶギミック[（5ch報告）](../sources/5ch-threads.md)
    - **朧車**: 車の目玉で魂を集めて結界を剥がすギミックあり[（5ch報告）](../sources/5ch-threads.md)
    - **扇子持ちの敵**: 他の敵にバフをかけるため優先して倒す[（5ch報告）](../sources/5ch-threads.md)
    - **大筒鬼（青い鬼）**: 即死級火力かつ精度が高い。見つけ次第最優先で処理[（5ch報告）](../sources/5ch-threads.md)

!!! tip "目潰し玉の有用性"
    攻城のボス戦でも**目潰し玉→闇討ち**のコンボが極めて強力。
    放心玉よりも目潰し玉を優先したい[（5ch報告）](../sources/5ch-threads.md)

!!! warning "試しの符「増援」"
    攻城で「増援」を選ぶと雪女3体が追加出現する場合あり。
    火のない状態では壊滅する。[（5ch報告）](../sources/5ch-threads.md)

## ボス共通の仕組み

- **黄色攻撃**: ボスの溜め強攻撃。HPの上の体幹ゲージを削り切るとキャンセル可能[（5ch報告）](../sources/5ch-threads.md)
- HP一定以下で攻撃パターンが変化（フェーズ移行）
- カメラワークや環境色が変わるのがフェーズ移行の目印
