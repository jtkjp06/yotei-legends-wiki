# 攻城

## 基本ルール

- **最大4人**で挑む大規模ミッション（PS公式JP）
- 奇譚の各セット（3話）をクリアすると対応する攻城が解放
- ラストに羊蹄六人衆のボス戦

## ボス一覧

| ボス | テーマ | 備考 |
|------|--------|------|
| 蜘蛛 | 時限弾・範囲攻撃 | 超範囲時限弾の回避が難所（5ch実測） |
| 鬼 | 風神雷神ギミック | 協力必須。風神3人推奨（5ch実測） |
| 狐 | 雪女＋冷気 | 火属性で戦う。火のない状態は絶望的（5ch実測） |
| 蛇 | 3色ギミック | 色合わせが分かれば最も低難度との報告 |

!!! warning "試しの符「増援」"
    攻城で「増援」を選ぶと雪女3体が追加出現する場合あり。
    火のない状態では壊滅する。（5ch実測）

## ボス共通の仕組み

- **黄色攻撃**: ボスの溜め強攻撃。HPの上の体幹ゲージを削り切るとキャンセル可能（5ch実測）
- HP一定以下で攻撃パターンが変化（フェーズ移行）
- カメラワークや環境色が変わるのがフェーズ移行の目印
