# モード別攻略 概要

冥人奇譚には**5つのゲームモード**があります。初めてプレイする際は「悪鬼成敗（奇譚）」から始めましょう。

## モード一覧

| モード | 内容 | 最大人数 | 解放条件 |
|--------|------|---------|---------|
| [悪鬼成敗（奇譚）](story.md) | メインストーリーミッション | 4人 | 最初から |
| [九死（サバイバル）](survival.md) | 波状攻撃を生き延びる | 4人 | 最初から |
| [攻城](incursion.md) | 敵拠点を攻略する | 4人 | <!-- TODO: 解放条件確認 --> |
| [レイド（大禍）](raid.md) | 高難度ボス戦 | 4人 | <!-- TODO: 解放条件確認 --> |
| [百鬼](hyakki.md) | 100体撃破チャレンジ | 4人 | <!-- TODO: 解放条件確認 --> |

!!! warning "要検証"
    各モードの解放条件は現在調査中です。

## 難易度について

各モードには複数の難易度が設定されています。高難度ほど良い報酬が手に入ります。

!!! tip "おすすめの進め方"
    まずはストーリーモード（奇譚）で基本操作を覚え、気レベルを上げてから九死や攻城に挑戦しましょう。

## 気レベルと報酬

各モードをクリアすることで**気（き）レベル**が上がり、より強力な装備が手に入るようになります。  
詳細は [気レベル上げ・周回](../guide/ki-farming.md) ガイドをご覧ください。
