# 忍（しのび）

忍は**高い機動力と奇襲能力**を持つトリッキーなクラスです。耐久力は低いですが、敵の隙を突く素早い攻撃と回避が得意です。使いこなすと非常に強力ですが、習熟が必要な上級者向けクラスです。

## 基本ステータス

| ステータス | 値 | 備考 |
|-----------|-----|------|
| HP        | 低  | クラス中最低 |
| スタミナ  | 最高| 連続回避が得意 |
| 攻撃力    | 中  | 奇襲時は高倍率 |
| 防御力    | 最低| 被弾は致命的 |
| 移動速度  | 最高| クラス中最速 |

!!! warning "要検証"
    上記の数値は概算です。正確な数値は調査中です。

## おすすめビルド

### 奇襲特化ビルド

!!! tip "立ち回りポイント"
    - 常に敵の死角を取ることを意識する
    - 「影走り」→ 背面攻撃 → 離脱のサイクルを繰り返す
    - 「暗殺」パッシブを最優先で取得しよう

### サポートビルド

!!! tip "サポートとしての役割"
    - 「毒霧」で敵の行動を制限しながら味方の攻撃チャンスを作る
    - 「分身の術」で敵のタゲを分散させる
