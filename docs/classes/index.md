# クラス攻略 概要

冥人奇譚には**4つのクラス**があります。それぞれ異なるプレイスタイルと役割を持っており、パーティの編成によって戦略が変わります。

## クラス比較表

| クラス | 役割 | 攻撃力 | 防御力 | 機動力 | 難易度 |
|--------|------|--------|--------|--------|--------|
| [侍](samurai.md) | タンク / ディーラー | ★★★ | ★★★★★ | ★★☆ | ★★☆ |
| [弓取](archer.md) | 遠距離アタッカー | ★★★★ | ★★☆ | ★★★ | ★★★ |
| [用心棒](mercenary.md) | 万能型ディーラー | ★★★★★ | ★★★ | ★★★ | ★☆☆ |
| [忍](shinobi.md) | サポート / 奇襲 | ★★★ | ★☆☆ | ★★★★★ | ★★★ |

## クラスの選び方

### 初心者におすすめ

**用心棒** はHP・攻撃力ともに高く、操作が比較的シンプルなため初心者にお勧めです。

### 上級者向け

**忍** は機動力が高い反面、耐久力が低いため立ち回りの上手さが求められます。使いこなすと非常に強力なクラスです。

### パーティ編成のポイント

!!! tip "バランスの良い構成"
    4人プレイでは「侍1 + 弓取1 + 用心棒1 + 忍1」のフルメンバー構成が最もバランスが取れています。
    各クラスが固有のスキルを持ち寄り、弱点を補い合えます。

!!! note "同クラスの重複"
    同じクラスを複数人が使用することも可能です。状況に応じて柔軟に対応しましょう。

## クラス固有スキル

各クラスはレベルアップによってスキルを習得します。スキルは**アクティブ**と**パッシブ**の2種類があります。詳細は各クラスのページをご覧ください。
