# 侍（さむらい）

侍はパーティの要となる**近接特化クラス**です。高い防御力とスタミナを活かして前線を維持しつつ、強力な刀術で敵を斬り倒します。

## 基本ステータス

| ステータス | 値 | 備考 |
|-----------|-----|------|
| HP        | 高  | クラス中最高 |
| スタミナ  | 中  | — |
| 攻撃力    | 中  | 刀のみ高い |
| 防御力    | 最高| — |
| 移動速度  | 中  | — |

!!! warning "要検証"
    上記の数値はゲーム内の表示に基づく定性的な評価です。正確な数値は現在調査中です。

## おすすめビルド

### 前衛タンクビルド

近接戦闘で敵の攻撃を引き付け、味方をサポートする構成です。

!!! tip "立ち回りポイント"
    - 「不動の構え」を大型敵やボスの強攻撃に合わせて使う
    - スタミナ管理を意識して無駄なローリングを避ける
    - 「残心」でスタミナを回復しながら継続して攻撃する
