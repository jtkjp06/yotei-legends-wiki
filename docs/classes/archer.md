# 弓取（ゆみとり）

弓取は**遠距離攻撃**を得意とするクラスです。弓と爆発物を駆使して、安全な距離から大ダメージを与えます。精密な照準と状況判断が求められる上級者向けクラスです。

## 基本ステータス

| ステータス | 値 | 備考 |
|-----------|-----|------|
| HP        | 中低 | — |
| スタミナ  | 中  | — |
| 攻撃力    | 高  | 遠距離時最大 |
| 防御力    | 低  | 接近されると危険 |
| 移動速度  | 中  | — |

!!! warning "要検証"
    上記の数値は概算です。正確な数値は調査中です。
