# 用心棒（ようじんぼう）

用心棒は**高い攻撃力と生存能力**を兼ね備えた万能型クラスです。シンプルな操作で強力な攻撃が可能なため、初心者にもお勧めです。

## 基本ステータス

| ステータス | 値 | 備考 |
|-----------|-----|------|
| HP        | 高  | — |
| スタミナ  | 高  | — |
| 攻撃力    | 最高| クラス中最高 |
| 防御力    | 中  | — |
| 移動速度  | 中  | — |

!!! warning "要検証"
    上記の数値は概算です。正確な数値は調査中です。
