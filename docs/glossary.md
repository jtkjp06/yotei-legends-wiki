# 用語集

## ゲーム内用語

| 用語 | 英語 | 意味 |
|------|------|------|
| 気 | Ki | 装備の平均レベル。キャラクターの総合力 |
| 格 | Rank | クラスの経験値レベル。スキル解放に関係 |
| 良品 | Common | 最低ランクの装備 |
| 名品 | Rare | 中ランクの装備 |
| 絶品 | Epic | 高ランクの装備 |
| 神品 | Legendary | 最高ランク。固有効果あり |
| 試しの符 | Challenge Card | 難易度を上げる代わりに報酬を増やす縛りカード |
| 受け流しの極意 | Perfect Parry | タイミングぴったりの受け流し |
| 闇討ち | Assassination | ステルスキル |
| 穢れ | Corruption | 紫の範囲ダメージ。一定時間で致死 |
| 霧隠れ | Vanish | 忍のステルス能力 |
| 八幡の怒り | — | 侍の奥義（範囲近接攻撃） |
| 内経の眼 | — | 弓取の奥義（ロックオン複数射撃） |
| 冥人の兵具 | Ghost Weapons | 苦無・煙玉等のサブ武器カテゴリ |
| 大岩加護 | — | 九死の拠点で購入できる加護 |
| 貢ぎ物 | — | 九死で敵を倒して得る勾玉（通貨） |

## コミュニティ用語

| 用語 | 意味 |
|------|------|
| パシャ撃ち | 長弓を引き絞らずに即撃ちする技法（ダメージ同じ） |
| 月キャン | 前作で使われたキャンセル技。今作では使用不可 |
| 寄生 | 推奨気より大幅に低い状態で高難度に参加すること |
| 床ペロ | 死亡して這いつくばっている状態 |
| 萎え抜け | クリア見込みがないと判断して途中退出すること |
| タタレタマシ | 前作の経験値稼ぎ手法。今作での類似手法は未確認 |
