# 装備システム

## 気（Ki）レベル

- 装備全スロットの平均値が「気レベル」になる（前作同様）
- 気レベルが高いほど、ドロップする装備のレベルも高くなる
- **現状の上限は110**（5ch報告。アプデで上限解放の可能性あり）

## 装備ランク

| ランク | ドロップ気レベル帯 | 入手先 |
|--------|---------------------|--------|
| 良品 | 低（刷3前後まで） | 赤銅〜白銀 |
| 名品 | 中（刷5前後まで） | 白銀〜黄金 |
| 絶品 | 高（気109のドロップ報告あり） | 黄金〜百鬼 |
| 神品（レジェンダリー） | 気レベルに依存しない | 黄金〜（低確率） |

!!! info "装備ランクと気レベルの関係"
    ドロップする装備の気レベルは、自分の現在の気レベルに依存します。絶品は気105以降で伸びが鈑化しますが、気109の絶品がドロップした報告もあります。[（5ch報告）](../sources/5ch-threads.md)
    神品はドロップ時の気レベルが固定ではなく、黄金以上であれば確率で出現します。

## 気の強化

- 素材を使って装備のレベルを上げられる
- **コストが安いのでケチらずやるべき**[（5ch報告）](../sources/5ch-threads.md)
- 強化してレベルを上げる→ドロップのレベルも上がる→効率的に成長
- 良品→名品のような「位を上げる」強化は大量の素材を消費するので注意

!!! tip "公式チュートリアルの助言は正しい"
    「最上級を手に入れるより武具強化した方がいい」というチュートリアルは正しい。
    序盤からコツコツ強化して平均気を上げていくのが最効率。[（5ch報告）](../sources/5ch-threads.md)

## 品格（特殊効果判定ランク）

!!! danger "厳選の核心"
    装備には **品格（特殊効果の強さを表すランク）** がある（D〜S）。
    **ドロップ時に固定** され、後から変更するには大量の素材が必要。

- 2つの判定があり、例えば「A-」と「B+」のように表示される
- 気レベルが低い装備でもA判定が出ることがある
- **分解する前に判定を確認すべき**（5ch報告で多くの人がA判定装備を捨てて後悔）

### 判定の上げ方

- 強化で判定を上げることは可能（C→C+のように段階的）
- ただし絶品の判定強化は**結晶200個**と高コスト[（5ch報告）](../sources/5ch-threads.md)
- 結晶200個 = 神品強化10回分 = 絶品強化100回分
- ADのような偏った装備より、BBのように平均の高い装備を強化する方が効率的

### 判定の上限

- Sで上限。片方がSになるとそれ以降はもう片方のランクしか上がらない
- 一部の判定（奥義回復など）はA手前とSで数値が変わらない場合あり[（5ch報告）](../sources/5ch-threads.md)

## 装備ドロップの仕様

- 装備は**クラス依存のドロップテーブル**がある
- 例：忍では鎖鎌が落ちやすいが、侍では落ちない
- 他クラスの装備も低確率で出現する場合あり[（5ch報告）](../sources/5ch-threads.md)
- 装備は全クラスで共有可能

詳しくは[ドロップアイテム一覧](drops.md)を参照。
