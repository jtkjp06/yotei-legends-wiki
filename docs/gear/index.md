# 装備システム

冥人奇譚の装備システムについて解説します。適切な装備を揃えることで、戦闘効率が大幅に向上します。

## 装備の種類

| 種類 | 部位 | 特徴 |
|------|------|------|
| 武器 | 武器スロット | 攻撃力・属性に影響 |
| 防具（上） | 胴スロット | 防御力・HP に影響 |
| 防具（下） | 脚スロット | 防御力・スタミナに影響 |
| お守り | アクセサリー | 特殊効果が付くことが多い |

!!! warning "要検証"
    装備の詳細なシステムは現在調査中です。

## レア度

装備にはレア度があり、高いほど強力な特殊効果を持ちます。

| レア度 | 色 | 特殊効果スロット |
|--------|-----|----------------|
| 普通 | 白 | 0〜1 |
| 珍しい | 緑 | 1〜2 |
| 貴重 | 青 | 2〜3 |
| 希少 | 紫 | 3〜4 |
| **神品** | **金** | **4〜5** |

## 神品（しんしな）とは

**神品**は最高レア度の装備です。固有の強力な特殊効果を持ち、エンドゲームコンテンツで必須となります。

詳しくは [神品一覧](legendary.md) ページをご覧ください。

## 特殊効果（パーク）

装備に付く特殊効果によって、キャラクターの能力を大きく変えることができます。

詳しくは [特殊効果・パーク](perks.md) ページをご覧ください。

## 気レベルと装備

**気（き）レベル**はキャラクターの総合的な装備強度を示す指標です。  
気レベルを上げるには、より高いレベルの装備を装備する必要があります。
